# ZeroADR 1.2.0rc1 Readiness

Status: implementation complete; ASB security targets passed, utility gate failed.

The release candidate adds opt-in multi-stage runtime gates while keeping
missing-policy behavior compatible with v1.1. The formal ASB v3 holdout
completed 100 attack and 100 paired clean cases in each of Baseline, Rules, and
Hybrid. Final Provider and workflow failures are zero. Two cache-only replays
made zero model calls and reproduced the same analysis hash.

task success is 54.5% versus Baseline 66.0%, a decline of 11.5 percentage
points. This misses the utility target by 1.5 points. The v3 holdout was not
used for tuning, and no post-holdout rule, prompt, threshold, or manifest
change was made.
